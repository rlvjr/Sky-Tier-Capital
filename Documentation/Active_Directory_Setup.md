## Overview

Active Directory Domain Services (AD DS) was implemented as the centralized identity and access management solution for the Sky Tier Capital infrastructure environment.

The purpose of this deployment is to simulate an enterprise Windows domain environment used to manage authentication, users, computers, and security policies.

---

## Environment Details

| Component | Configuration |
|:---:|:---:|
| Organization | Sky Tier Capital |
| Domain Name | SkyTierCapital.local |
| Operating System | Windows Server 2022 |
| Server Role | Domain Controller |
| Directory Service | Active Directory Domain Services |
| Virtualization Platform | Oracle VirtualBox |

---

## Implementation Completed

The following Active Directory deployment tasks have been completed:

Installed Active Directory Domain Services (AD DS)

Promoted Windows Server 2022 to a Domain Controller

Created the SkyTierCapital.local domain environment

Verified successful domain authentication using the domain administrator account

---

## Active Directory Purpose

Active Directory provides centralized management of:

- User authentication
- Computer accounts
- Security groups
- Access permissions
- Organizational structure
- Group Policy enforcement
