# Project Overview

## Sky Tier Capital Enterprise IT Infrastructure Simulation

</div>

## Overview

Sky Tier Capital is a simulated enterprise IT infrastructure environment designed to demonstrate practical system administration, networking, and cybersecurity skills.

The project represents the internal technology environment of a fictional financial trading organization that requires secure, reliable, and controlled access to business resources.

This lab was created to replicate common responsibilities performed by IT Support Specialists, System Administrators, and Security Analysts.

---

## Business Scenario

Sky Tier Capital operates as a financial trading organization that depends on secure technology systems to support daily operations.

The simulated IT department is responsible for:

* Managing user identities and access
* Maintaining Windows Server infrastructure
* Supporting employee workstations
* Implementing security controls
* Troubleshooting technical issues
* Documenting system changes and incidents
