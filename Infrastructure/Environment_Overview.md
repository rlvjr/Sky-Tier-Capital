## Sky Tier Capital Infrastructure Environment

</div>

## Overview

Sky Tier Capital is a simulated enterprise IT infrastructure environment designed to demonstrate the deployment, configuration, and management of business technology systems.

The environment represents a fictional financial trading organization requiring secure authentication, reliable services, controlled access, and documented IT operations.

---

## Infrastructure Purpose

The purpose of this environment is to simulate common responsibilities performed by IT Support Specialists and System Administrators, including:

- Server deployment
- Identity management
- Network service configuration
- User administration
- Security management
- Troubleshooting and documentation

## Core Infrastructure Components

### Windows Server

Windows Server 2022 provides the foundation of the environment and hosts critical enterprise services.

Responsibilities:

- Domain services
- Authentication
- Network management
- Administrative tools

---

### Active Directory Domain Services

Active Directory provides centralized identity and access management.

Responsibilities:

- User authentication
- Computer management
- Security groups
- Access control

---

### DNS Services

DNS provides internal name resolution required for domain functionality.

Responsibilities:

- Domain name resolution
- Server communication
- Active Directory service discovery

---

### Group Policy Management

Group Policy provides centralized configuration management.

Responsibilities:

- Security settings
- User restrictions
- System configuration
- Administrative policies
